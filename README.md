# A boilerplate of latest MEVN technologies
You must specify your own .env file to connect to your MongoDB database. 

Installed Plugins
- Babel
- Router
- Pinia
- Vue Axios
- Bulma
- Typescript

# Get Started

Install dependencies

root folder(server)

``` npm install ```

client

``` cd client && npm install ```

# Command Scripts
Run only in the root folder


Server only

``` npm run server ```

Client only 

``` npm run client ```

Development server

``` npm run dev ```

Production server 

``` npm run start ```
